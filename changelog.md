# Update to 1.20.4

# Additions

- Shape Group HUD: The shape groups of a spell will now be shown in the top left corner (by default), with the selected shape group highlighted
- Light block: The Light spell component now uses its own block instead of the vanilla mapmaker light

# Changes and Fixes

- Fix the magic unlocking not always working properly
- Fix magic xp not scaling with stronger spells, this also now makes leveling considerably faster
- Fix damage not always applying correctly, and allow multiple damage spells to properly stack
- Fix various small issues with the rituals
